Gestion du délai d’exécution en JavaScript avec setTimeout() et setInterval()

Parfois, on ne voudra pas exécuter une fonction immédiatement mais plutôt dans un certain temps ou on pourra encore vouloir exécuter une fonction plusieurs fois avec un intervalle de temps défini entre chaque exécution.

Le JavaScript met à notre disposition deux méthodes pour gérer le délai d’exécution d’un code : les méthodes setInterval() et setTimeout() qui sont des méthodes qui vont être implémentées par Window dans un contexte Web.

La méthode setTimeout()

setTimeout() permet d’exécuter une fonction ou un bloc de code après une certaine période définie (à la fin de ce qu’on appelle un « timer »).
On passe 2 arguments à cette méthode : le nom de la fonction à exécuter et un nombre en millisecondes qui représente le délai d’exécution de la fonction.
On pourra aussi passer des arguments facultatif

Notez que la méthode setTimeout() renvoie un entier positif à l’issue de son exécution qui va identifier la timer créé par l’appel à setTimeout(). On va ainsi pouvoir utiliser cet entier pour annuler l’exécution de la fonction à laquelle on a ajouté un délai avec clearTimeout().

En pratique

let btn1 = document.getElementById('btn1');
btn1.addEventListener('click', message);

function message() {
setTimeout(alert, 2000, 'Message d\'alerte après 2 seconde'); - // Attention, ici 3 arguments (la fonction alert, le temps, et le message de la fonction alert())
}

La méthode clearTimeout()

La méthode clearTimeout() va nous permettre d’annuler ou de supprimer un délai défini avec setTimeout() (et donc également d’annuler l’exécution de la fonction définie dans setTimeout()).
Pour que cette méthode fonctionne, il va falloir lui passer en argument l’identifiant retourné par setTimeout().

En pratique :

let btn2 = document.getElementById('btn2');
let timeoutId; - // on crée cette variable pour s'en servir avec clearTimeou()

btn2.addEventListener('click', messageBtn2);
function messageBtn2(){
timeoutId = setTimeout(alert, 2000, 'Btn2 message après 2s'); }; - // on récupère le résultat renvoyé par setTimeout() dans la variable timeoutId.

let btn3 = document.getElementById('btn3');
btn3.addEventListener('click', stopDelai);

function stopDelai() {
clearTimeout( timeoutId ); - // on fait un clearTimeout() sur la variable timeoutId
}

La méthode setInterval()

setInterval() permet d’exécuter une fonction ou un bloc de code en l’appelant en boucle selon un intervalle de temps fixe entre chaque appel.
Elle prend en arguments le bloc de code à exécuter en boucle et l’intervalle entre chaque exécution exprimé en millisecondes.
En arguments facultatifs : les arguments de la fonction qui doit être exécutée en boucle.

setInterval() renvoie un entier positif qui va servir à identifier un appel à setInterval() et nous permettre d’annuler l’intervalle de répétition avec clearInterval().

En pratique :

let btn4 = document.getElementById('btn4');
let btn5 = document.getElementById('btn5');
let p1 = document.getElementById('p1');
let intervalHologeId;

btn4.addEventListener('click', afficheHeure);

function afficheHeure() {
intervalHologeId = setInterval(function () {
let d = new Date();
- // on appelle l'objet Date() p1.innerHTML = d.toLocaleTimeString(); - // ici on se sert de tolocaleTimeString pour afficher l'horolge
}, 1000);
};

btn5.addEventListener('click', stopHorloge);

function stopHorloge() {
clearInterval(intervalHologeId);
};
