Supprimer, cloner, déplacer un nœud dans le DOM

Cloner ou remplacer un nœud dans le DOM

  • cloneNode() de Node : renvoie une copie du nœud sur lequel elle a été appelée. Il faudra indiquer où doit se positionner le clone avec une méthode (par ex : insertAdjacentElement()).
    Cette méthode prend un booléen en argument. Si la valeur passée est true, les enfants du nœud seront également clonés. Si on lui passe false, en revanche, seul le nœud spécifié sera cloné.
    Par défaut, cette méthode copie également le contenu du nœud cloné.

    Un noeud d'id="p1"
    Du texte suite à un br

    let li1 = document.getElementById('li1'); on cilbe la balise li d'id="li1"
    let p1 = document.getElementById('p1'); //on cible l'élément "p1"
    let cloneP1 = p1.cloneNode(true); ; ∕∕on stocke dans la variable cloneP1 ce que l'on veut cloner (p1 stocké dans la variable p1)
    li1.appendChild(cloneP1); // on clone en tant que dernier enfant de li1, cloneP1 (et ses enfants)

  • replaceChild() de Node : va remplacer un certain nœud par un autre.
    Cette méthode va prendre en arguments le nœud de remplacement et le nœud qui doit être remplacé.
    Noter: que si le nœud de remplacement existait déjà dans le DOM, il sera d’abord retiré de son emplacement d’origine.

    let li2 = document.getElementById('li2'); //on cible li2
    let p2 = document.getElementById('p2'); //on cible l"élément à remplacer
    li2.replaceChild(newP, p2); //on indique que l'on remplace dans l12, p2 par newP (newP remplace p2 dans li2)

Supprimer un nœud du DOM

  • remove() du mixin ChildNode() : permet tout simplement de retirer un nœud de l’arborescence et qui dispose aujourd’hui d’un bon support par les navigateurs (cette façon de faire est plus récente que la précédente).

    1er paragraphe id="sp1"

    2eme paragraphe id="sp2"

    3e paragraphe id="sp3"

    let sp2 = document.getElementById('sp2');
    sp2.remove();

  • removeChild() de Node : va supprimer un nœud enfant passé en argument d’un certain nœud parent de l’arborescence du DOM et retourner le nœud retiré.

    let li3 = document.getElementById('li3'); //on cible li3 comme parent
    let p3 = document.getElementById('p3'); //on cible l'élément p3 à supprimer
    let eltDel = li3.removeChild(p3); //on indique que l'on supprime p3 de li3 et renvoie le noeud supprimé
    console.log(eltDel); // renvoie l'élément supprimé et son contenu
